Tīmekļa aplikācija individuālā budžeta plānošanai

Tā izveidota ar React, Vite, Tailwind CSS, Chart.js, PHP un Laravel tehnoloģijām.
Ir iebūvēta autentifikācija: e-pasta adrese un parole. Funkcionalitāte sastāv no informācijas panelī esošām CRUD operācijām-ierakstu izveidošana, nolasīšana, atjaunināšana, dzēšana. Papildus ir grafisks attēlojums, ierakstu dalījums pa atsevišķām lapām, ierakstu filtrēšana, ienākumu un izdevumu salīdzinājums, profila lapa, paskaidrojošā lapa.
